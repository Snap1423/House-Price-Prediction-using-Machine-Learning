# House Price Prediction (ML Project)

This is my first machine learning project where I built a model to predict house prices using real-world data.

## What did I do 
* Data cleaning (handled missing values)
* Feature selection (X and y split)
* Train/test split
* Built multiple models:

  * Linear Regression
  * Decision Tree
  * Random Forest
* Evaluated using RMSE
* Visualized results using graphs

## Results:

* Linear Regression RMSE: ~70k
* Decision Tree RMSE: ~68k
* Random Forest RMSE: ~49k

## Key Learnings:

* Importance of data cleaning
* Understanding overfitting
* Comparing models
* Using visualization for analysis
